# Developer Interview Handbook

Manual vivo de formación y preparación técnica para la transición profesional de **QA a Frontend, Mobile o Full Stack Developer**.

## Objetivo

Este repositorio es la fuente oficial de conocimiento del proyecto. El contenido combina:

- preguntas reales de entrevistas;
- respuestas recomendadas;
- explicaciones teóricas en modo profesor/tutor;
- repreguntas y escenarios de presión;
- ejercicios para responder con palabras propias;
- feedback de entrevistas simuladas;
- documentación técnica de proyectos reales.

## Proyectos principales

- **Monedero:** aplicación mobile de finanzas personales publicada en App Store.
- **Editor Académico APA:** editor web para documentos universitarios con normas APA 7.
- **Prode:** plataforma web de pronósticos de fútbol.
- **Picadito:** aplicación colaborativa para organizar partidos de fútbol amateur.

## Método de estudio

Cada tema sigue esta estructura:

1. Pregunta del entrevistador.
2. Qué intenta evaluar.
3. Respuesta breve.
4. Respuesta desarrollada.
5. Explicación del profesor.
6. Alternativas y tradeoffs.
7. Repreguntas probables.
8. Errores comunes.
9. Ejercicio práctico.
10. Nivel de dominio.
